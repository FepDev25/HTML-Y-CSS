Box Model - Modelo de Caja

Concepto que describe como se disenan y como se deberian colocar los elementos de html en una pagina web. Todo en html se considera como una caja rectangular.

Propiedades

  1. Contenido: Area en donde se muestra el contenido, como texto o imagenes.
  2. Padding (o relleno): Espacio entre el contenido y el borde del elemento.
  3. Border: Linea que rodea tanto el contenido como el padding. Se puede controlar el estilo, el grosor y el color.
  4. Margen: Espacio al rededor de la pagina que esta fuera del borde. Separa el elemento de otros elementos en la pagina.
